Обложки карт для голосования на «Главной».

Положи сюда 6 картинок (свои скриншоты карт из игры, горизонтальные, лучше 16:9,
одинаково кадрированные — тогда сетка будет ровной). Имена строго такие:

  zero_dam.jpg      — Плотина Зеро (Zero Dam)
  space_city.jpg    — Космический город (Space City)
  layali_grove.jpg  — Роща Лаяли (Layali Grove)
  brakkesh.jpg      — Брекеш (Brakkesh)
  tide_prison.jpg   — Тюрьма «Прилив» (Tide Prison)
  az3.jpg           — АЭС AZ3 (Nuclear Power Plant)

Пока файла нет — карточка показывает стилизованную заглушку (градиент), ничего не ломается.
Форматы: .jpg (можно .png/.webp, но тогда поправь расширения в MP_MAPS в docs/index.html).
Рекомендуемый размер: ~800×600 и меньше 300 КБ на файл, чтобы «Главная» грузилась быстро.

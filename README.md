# DF Аукцион — Telegram-бот экономики Delta Force

Цены аукциона, вотчлист, алерты и собственная копилка истории цен.
Данные: открытый API deltaforceapi.com (глобальный сервер = тот, где играет РФ).

## Что умеет (MVP)

- Поиск предмета по названию — русские алиасы и транслит («дигл» → Desert Eagle)
- Карточка цены: текущая цена, референс, изменение за 24ч/7д, мини-график, сигнал дёшево/дорого
- ⭐ Вотчлист и 🔔 алерты («сообщи, когда упадёт ниже X»)
- /wipe — сколько дней до конца сезона
- Сборщик: каждые 15 минут пишет цены отслеживаемых предметов в свою базу (df.sqlite3)

## Запуск (Windows)

1. Получи токен бота: в Telegram открой @BotFather → /newbot → придумай имя → скопируй токен
2. Скопируй `.env.example` в `.env` и вставь токен
3. В папке проекта:

```powershell
python -m venv venv
venv\Scripts\pip install -r requirements.txt
venv\Scripts\python bot.py
```

Первый запуск скачает справочник предметов (~минута). Дальше бот отвечает в Telegram.

## Структура

| Файл | Назначение |
|---|---|
| bot.py | Хендлеры, карточки, фоновые задачи |
| api.py | Клиент deltaforceapi.com |
| db.py | SQLite: пользователи, вотчлисты, алерты, история цен |
| aliases.py | Русские алиасы + транслит для поиска |
| config.py | Настройки, .env |
